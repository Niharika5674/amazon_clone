1. Amazon Clone (HTML & CSS)

  This project is a beginner-level clone of the Amazon homepage created using only HTML and CSS. 
  The purpose of this project is to practice basic web development skills such as page layout, styling, and responsive design.

2. About the Project
  The website replicates the basic design and layout of the Amazon homepage, including the navigation bar, product sections, and footer. 
  It focuses on understanding how HTML structures a webpage and how CSS is used to style and position elements.

3. Technologies Used
  - HTML
  - CSS

4. Purpose
The goal of this project is to:
- Improve understanding of HTML and CSS
- Practice creating real-world website layouts
- Build a beginner portfolio project
